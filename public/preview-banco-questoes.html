✗  Resposta incorreta.
Gabarito
✓  Alternativa C — 200 cm²
Resolução
Seja largura = x e comprimento = 2x.
Perímetro: 2(x + 2x) = 40 → 6x = 40 → x = 20/3 cm
Comprimento = 40/3 cm
Área = x × 2x = 2x² = 2 × (20/3)² = 800/9 ≈ 88,9 cm²
